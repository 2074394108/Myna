## Data Structure

数据集中包含下面三种行为的传感器数据：

- Walking
- In Vehicle
- Still

使用到的传感器类型

- 加速度
- 陀螺仪
- 重力加速度

### 数据格式

	accelerometerX,accelerometerY,accelerometerZ,GyroscopeX,GyroscopeY,GyroscopeZ,GravityX,GravityY,GravityZ

### 其他说明

训练数据的采样频率为 50Hz, 512 条记录为一组保存在一个文件中。
